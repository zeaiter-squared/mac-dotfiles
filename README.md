For the most part everything is handled by the `setup.sh` script. There are several items that need to be
addressed manually.

1. Clone this repo (a popup will appear to automatically install `git`).
2. Run `setup.sh`.
3. Configure scroll-reverser to only work on mice, not trackpads.
4. Download and install Docker. Configure this to start on login.
5. Download and install Postman.
6. Download and install Notion.
7. Setup Git over SSH to Azure DevOps.
8. Setup the capslock script to run at login. This should be handled by `setup.sh` but it may need to be
   enabled in System Settings > Login Items.
9. Set wallpaper.
10. Set theme to dark.
11. Setup desktop behaviour, 5 desktops mapped to option+1--5 (respectively).
12. Disable all Mission Control settings in System Settings.
13. Use function keys as default.
13. Login on Microsoft Edge.
14. Reverse Ctrl and fn key.
15. Reverse Option and Command key.
